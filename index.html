verified_user Política de Privacidade

Última atualização: Novembro de 2025

A Cronatech Sistemas ("nós", "nosso") desenvolveu o aplicativo Drivio como um serviço comercial. Esta política descreve nossas práticas de dados.

warning Divulgação Importante: Serviços de Acessibilidade

O aplicativo Drivio utiliza a Android Accessibility API.

  • Finalidade: Ler informações da tela (valor, distância, tempo) de apps de transporte para calcular a rentabilidade em tempo real.
  • Segurança: NÃO coletamos senhas, dados bancários ou conversas pessoais.
  • Armazenamento: Os dados da tela são processados localmente e descartados; não são enviados para servidores externos para fins de mineração.

1. Coleta e Uso de Informações

Para proporcionar uma melhor experiência, podemos solicitar certas informações de identificação (como e-mail para login). As informações retidas serão usadas conforme descrito nesta política.

2. Dados de Localização

O Drivio pode coletar dados de localização em segundo plano para monitorar o trajeto da corrida e calcular custos por KM, mesmo quando o app está minimizado.

3. Segurança

Valorizamos sua confiança. Utilizamos protocolos modernos de criptografia para proteger seus dados. Nenhum método de transmissão pela internet é 100% seguro, mas nos esforçamos para usar meios comercialmente aceitáveis de proteção.

4. Contato

Dúvidas sobre esta política? Entre em contato: contato@cronatech.com.br
